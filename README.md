# RetroAI – Python Smart Chatbot

RetroAI is a Python-based AI chatbot that can learn, remember, and respond intelligently.
It also supports math solving, QR code generation, and voice output.

# Features
- Learning & memory using JSON
- Math solver (SymPy)
- NLP similarity matching
- Text-to-Speech
- QR code generation
- TFT knowledge support

# Project Structure
- retro_ai.py → Main chatbot logic
- update_tft.py → Update TFT knowledge
- knowledge.json → AI memory database

# Built With
- Python
- SymPy
- Scikit-learn
- pyttsx3
- qrcode
